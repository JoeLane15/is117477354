Cloud9 and Platform as a Service (PaaS)

Cloud9 IDE is an online integrated development environment, published as open source from version 3.0. It supports hundreds of programming languages, including C, C++, PHP, Ruby, Perl, Python, JavaScript with Node.js, and Go. Cloud 9 was founded in 2010 and was eventually bought by Amazon.com in 2016

Platform as a Service (PaaS):

Platform as a Service (PaaS), is a type of a cloud computing service. It provides a platform allowing customers to develop, run and manage applications without the complexityy of building and maintaining the infrastructure typically associated with developing and launching an app.PaaS can be delivered in three ways: as a public cloud service from a provider, where the consumer controls software deployment with minimal configuration options, and the provider provides the networks, servers, storage, operating system (OS), middleware (e.g. Java runtime, .NET runtime, integration, etc.), database and other services to host the consumer's application; as a private service (software or appliance) inside the firewall; or as software deployed on a public infrastructure as a service.
