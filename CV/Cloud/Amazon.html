Amazon Web Services and Infrastructure as a service

Amazon.com is an ecommerce and cloud service company. Founded in 1994 by Jeff Bezos, Amazon started life as an online bookstore but has since diversified to nearly all facets of technology

Infrastructure as a service:

Infrastructure as a service (IaaS), are the essential building blocks for cloud IT and allow access to data stroage space, network features and computers, virtual or dedicated hardware. High level of flexibility and management control over your IT resources are prominent features in Infrastructure as a Service systems.


Amazon Web Services or AWS is the cloud service platform provided by Amazon.com. Database storage, Compute power, content delivery and other functionalities are all services provided by AWS. When using AWS cloud products and solution you can create applications with increased flexiblity, scalability and reliability
