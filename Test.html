Platform as a Service (PaaS):

Platform as a Service (PaaS), is a type of a cloud computing service. It provides a platform allowing customers to develop, run and manage applications without the complexityy of building and maintaining the infrastructure typically associated with developing and launching an app.PaaS can be delivered in three ways: as a public cloud service from a provider, where the consumer controls software deployment with minimal configuration options, and the provider provides the networks, servers, storage, operating system (OS), middleware (e.g. Java runtime, .NET runtime, integration, etc.), database and other services to host the consumer's application; as a private service (software or appliance) inside the firewall; or as software deployed on a public infrastructure as a service.

Google Drive was founded in 2012 as a file storage and synchronization serivce developed by Google.


How it Works:

This service allows users to store their files on their servers, synchronize files across devices, and share files. Users can also avail of Google Docs, Sheets and Slides and an Office suite. These features allow collaborative editing of documents, spreadsheets, presentations, drawings, forms and more. The user get 15 gigabytes of free storage, with up to 30 terabytes offered through optional paid plans.

Dropbox is a file hosting serice that was founded by Drew Houstin and Arash Ferdowsi in 2007. Dropbox offers cloud storage, file synchronization, personal cloud and client software.


How it Works:

The Dropbox software enables users to drop any file into a designated folder. The file is then automatically uploaded to Dropbox's cloud-based service and made available to any other of the user's computers and devices that also have the Dropbox software installed, keeping the file up-to-date on all systems. When a file in a user's Dropbox folder is changed, Dropbox only uploads the pieces of the file that have been changed, whenever possible.
